# ALPINE Error Codes

ALPINE defines structured error codes:

### Discovery Errors
- DISCOVERY_INVALID_SIGNATURE
- DISCOVERY_NONCE_MISMATCH
- DISCOVERY_UNSUPPORTED_VERSION

### Handshake Errors
- HANDSHAKE_SIGNATURE_INVALID
- HANDSHAKE_KEY_DERIVATION_FAILED
- HANDSHAKE_TIMEOUT
- HANDSHAKE_REPLAY

### Session Errors
- SESSION_EXPIRED
- SESSION_INVALID_TOKEN
- SESSION_MAC_MISMATCH

### Control Errors
- CONTROL_UNKNOWN_OP
- CONTROL_PAYLOAD_INVALID
- CONTROL_UNAUTHORIZED

### Streaming Errors
- STREAM_BAD_FORMAT
- STREAM_TOO_LARGE
- STREAM_UNSUPPORTED_CHANNEL_MODE
